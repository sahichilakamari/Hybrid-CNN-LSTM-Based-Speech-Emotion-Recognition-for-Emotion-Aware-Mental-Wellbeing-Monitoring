# 🎤 Speech Emotion Recognition (SER) System

A deep learning-based Speech Emotion Recognition system that classifies human emotions from audio using a **CNN + LSTM** architecture.

---

## 🚀 Features

- 🎧 Audio-based emotion prediction
- 🧠 CNN + LSTM model
- 📊 Supports Mel Spectrogram + MFCC features
- 🌐 REST API using FastAPI
- 📈 Class imbalance handling (class weights)
- ⚡ Optimized for small datasets
